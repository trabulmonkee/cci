# cci

## Description ##
Quick repo to explore out CircleCI and Cypres test exeuction

Setup / Prequisities
1. Clone the repo
2. Install Node Version Manager [nvm](https://github.com/nvm-sh/nvm)
3. From root repo directory execute: `npm install`
4. Execute: `npm run cy:test`
5. Have fun!
